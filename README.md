# archive-cnh5-baccarat

## 项目简介

`archive-cnh5-baccarat` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为静态 HTML 内容集合的存储与共享空间。

## 目录结构

```
archive-cnh5-baccarat/
├── pages/           # 存放归档的 HTML 页面文件
├── assets/          # 页面引用的静态资源（图片、样式、脚本等）
└── README.md        # 本说明文件
```

- **pages/**：所有归档的独立 HTML 页面均放置于此目录下，每个页面为一个独立文件。
- **assets/**：页面所需的公共或私有资源文件，按需组织。

## 页面归档说明

- 本仓库中的 HTML 页面均为独立归档，彼此之间无直接关联。
- 页面内容可能包含不同时期、不同来源的静态信息，仅供存档和参考。
- 归档页面不保证与原始来源完全一致，也不代表任何立场或观点。
- 仓库不提供任何形式的动态服务或后端支持，所有页面均为纯前端静态内容。

## 使用方式

您可以通过以下方式访问本仓库中的页面：

1. 将仓库克隆至本地，使用浏览器直接打开 `pages/` 目录下的 HTML 文件。
2. 若仓库启用了 GitHub Pages，可通过对应 URL 访问 `pages/` 目录下的文件（具体 URL 请参考仓库设置）。
